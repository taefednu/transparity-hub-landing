## 🔧 Что нужно заполнить:

### 1. Команда (`frontend/src/components/TeamSection.tsx`)

Откройте файл и заполните для каждого участника:

```typescript
{
  name: "Имя Фамилия",
  role: "Конкретная роль (Product Lead, Data Scientist, Backend Developer, Frontend Developer)",
  skills: "Конкретные навыки и технологии (например: Python, ML, FastAPI, React, TypeScript)",
  experience: "1-2 предложения о предыдущем опыте и проектах",
  links: {
    linkedin: "https://linkedin.com/in/...",
    github: "https://github.com/...",
    telegram: "https://t.me/..."
  }
}
```

**Важно:** Заполните реальные данные для всех 4 участников!

### 2. Почему ваша команда (`frontend/src/components/WhyUsSection.tsx`)

Откройте файл и замените плейсхолдеры на реальные буллеты:

- Опыт команды в продуктовой разработке / финтехе / банковских проектах
- Понимание рынка Узбекистана и боли МСБ
- Опыт в ML/AI и интеграциях
- Мотивация и видение
- Предыдущие стартапы / хакатоны / достижения

**Пример:**
```typescript
<p>
  <strong>Команда имеет опыт разработки финтех-продуктов для банков Узбекистана...</strong>
</p>
```
