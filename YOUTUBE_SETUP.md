# Настройка YouTube видео с доступом только по ссылке

## Шаги для загрузки видео на YouTube

### 1. Загрузите видео на YouTube

1. Зайдите на [YouTube Studio](https://studio.youtube.com)
2. Нажмите **"Создать"** → **"Загрузить видео"**
3. Выберите файл `demo/Vortex AI500.mp4` или `frontend/public/demo-video.mp4`
4. Дождитесь загрузки

### 2. Настройте приватность (ВАЖНО!)

В настройках видео выберите:
- ✅ **"Не в списке" (Unlisted)** - видео будет доступно только по прямой ссылке
  - Не отображается в поиске YouTube
  - Не видно на вашем канале
  - Доступно только по прямой ссылке
  - Можно встроить на сайт через iframe

- ❌ **"Приватное" (Private)** - НЕ используйте, так как приватные видео нельзя встроить на сайт

**Рекомендуется:** Используйте **"Не в списке"**, так как это позволит встроить видео на сайт.

### 3. Получите ID видео

После загрузки YouTube даст вам URL вида:
```
https://www.youtube.com/watch?v=VIDEO_ID
```

Скопируйте `VIDEO_ID` (например, `dQw4w9WgXcQ`)

### 4. Обновите код

Код уже обновлен в `frontend/src/pages/DemoPage.tsx`. Просто замените `YOUR_VIDEO_ID` на ваш ID видео.

Например, если ваш URL: `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
То ID будет: `dQw4w9WgXcQ`

Замените в коде:
```tsx
src="https://www.youtube.com/embed/YOUR_VIDEO_ID?rel=0&modestbranding=1"
```

На:
```tsx
src="https://www.youtube.com/embed/dQw4w9WgXcQ?rel=0&modestbranding=1"
```

## Важные настройки YouTube

### Параметры iframe:
- `rel=0` - не показывать похожие видео в конце
- `modestbranding=1` - скрыть логотип YouTube (частично)

### Дополнительные параметры (опционально):
- `autoplay=1` - автовоспроизведение (требует `mute=1`)
- `loop=1` - зацикливание
- `controls=0` - скрыть элементы управления
- `start=30` - начать с 30 секунды

Пример с дополнительными параметрами:
```tsx
src="https://www.youtube.com/embed/YOUR_VIDEO_ID?rel=0&modestbranding=1&controls=1"
```

## Безопасность

✅ **"Не в списке" (Unlisted)** - идеально для демо:
- Не видно в поиске YouTube
- Не видно на канале
- Доступно только по прямой ссылке
- Можно встроить на сайт

❌ **"Приватное" (Private)** - НЕ подходит:
- Нельзя встроить на сайт
- Только вы сможете видеть
